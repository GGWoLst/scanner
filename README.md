# Network Scanner (Go)

Высокопроизводительный асинхронный сканер для анализа сетевой связности, поиска рабочих IP в условиях DPI блокировок.

## Особенности
- **Многопоточность**: проверка тысяч IP параллельно (горутины).
- **Пинг 1.1.1.1**: автоматическая остановка сканирования, если `1.1.1.1` становится доступен (означает, что блокировки или вайтлисты отключены).
- **TLS Handshake**: выявление DPI (через TCP RST) и отброшенных пакетов (Timeout).
- **MaxMind GeoIP**: автоматическое определение ASN и провайдера.
- **Web-панель**: темная тема, отображение прогресса в реальном времени.
- **Экспорт**: генерация `results.csv` и `results.json` на лету.

## Установка и запуск

Скрипт поддерживает два режима работы:
- **CLI (по умолчанию)**: выводит прогресс прямо в консоль и завершает работу по окончании.
- **Web-режим**: поднимает встроенный веб-сервер на порту 8080 для визуального мониторинга (флаг `--web`).

### Вариант 1: Docker (Рекомендуемый, запускается в Web-режиме)
Убедитесь, что у вас установлен Docker и Docker Compose.
1. Поместите нужные файлы списков (`.txt` с IP или подсетями) в директорию `cidr/`. Вы можете добавлять туда сколько угодно файлов.
2. Поместите файлы базы данных MaxMind (минимум `GeoLite2-ASN.mmdb`) в папку `geo/`.
3. Создайте пустые файлы для результатов, чтобы Docker корректно их примонтировал:
   ```bash
   touch results.csv results.json
   ```
4. Запустите контейнер:
   ```bash
   docker-compose up -d --build
   ```
5. Откройте веб-панель в браузере: `http://localhost:8080`.

### Вариант 2: Локальный запуск (требуется Root / Администратор)
1. Установите Go.
2. Загрузите зависимости:
   ```bash
   go mod tidy
   ```
3. Соберите бинарный файл:
   ```bash
   go build -o scanner .
   ```
4. Запустите скрипт (с правами root в Linux/WSL или Администратора в Windows):
   - **CLI-режим (в консоли):**
     ```bash
     sudo ./scanner
     ```
   - **Web-режим:**
     ```bash
     sudo ./scanner --web
     ```
     Затем откройте панель управления: `http://localhost:8080`.
